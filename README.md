# Netflix Movies and TV Shows Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python. It includes data loading, data cleaning, exploratory data analysis (EDA), data visualization, and key insights.

## Objectives
- Load and inspect the dataset.
- Clean missing values and prepare the data.
- Perform exploratory data analysis using Pandas.
- Create visualizations to understand trends.
- Generate insights from the dataset.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## Project Workflow
1. Load and Inspect Data
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights and Recommendations

## Key Insights
- Movies are more numerous than TV Shows.
- The United States contributes the highest number of titles.
- Netflix content has increased significantly after 2015.
- Most titles are recent releases.
- Data visualization helped identify important trends and patterns.
